
## Set Static IP  

Find the network interface, then edit the newtork config
```console
ip a
sudo nano /etc/netplan/01-netcfg.yaml
```

```yaml
network:
  version: 2
  renderer: networkd
  ethernets:
    ens18:
      dhcp4: no
      addresses:
        - 192.168.1.211/24
      routes:
        - to: default
          via: 192.168.1.1
      nameservers:
        addresses: [129.168.1.200,1.1.1.1,8.8.8.8]
```
```console
sudo chmod 600 /etc/netplan/01-netcfg.yaml
sudo netplan apply
ip a
```
